CutieFont

by Pixel Coder (beckiergb.deviantart.com) with artwork by BlackGryph0n, SilverVectors, ParclyTaxel, TheSmilingPony, ArcticKiwi, ZuTheSkunk, SirCxyrtyx, KatiePox, dipi11, uxyd, and Durpy.

CutieFont is a TrueType font with embedded color images, like emoji fonts, except instead of emoji it contains Cutie Marks! The version named CutieFont.ttf.sbix.ttf works only with Mac OS X 10.8 Mountain Lion or later; for details on that version, read the file CutieFont.rtf instead. The version named CutieFont.ttf.svg.ttf will work with Mozilla Firefox and will soon work with Adobe CC and Windows 10; for details on that version, read this file (CutieFont.html).

The font was generated by FontForge, but since FontForge does not support embedded color images, they had to be injected into the generated font using Kreative BitsNPicas.

CutieFont contains artwork by BlackGryph0n, SilverVectors, ParclyTaxel, TheSmilingPony, ArcticKiwi, ZuTheSkunk, SirCxyrtyx, KatiePox, dipi11, uxyd, and Durpy, and is released under Creative Commons Attribution Non-Commercial Share-Alike.
